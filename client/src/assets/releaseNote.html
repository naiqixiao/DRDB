Release Notes

Schedule Appointments Enhancements

  • Partial Update Feature: Capability to modify individual appointments within a schedule, beneficial for scheduling subsequent visits.
  • Direct Reschedule Functionality: Directly reschedule appointments to a new date and time, ideal for specific rescheduling needs.
  • Multiple Studies Per Visit: Supports booking multiple studies per visit, with each represented as separate Google Calendar events.

Testing Rooms Feature

  • Create testing rooms via the Settings page, each linked to an independent Google Calendar.
  • Associate studies with a testing room on the Study Management page; appointments will appear in the relevant Google Calendar.
  • Default to the main calendar if no specific testing room is set up.

Email System Enhancements

  • Email Tags: Organize emails by study name. Automatic in Gmail, manual format #%#study.name#%# for others.
  • Email Templates: New follow-up and reschedule email templates accessible via the Mail icon on scheduling pages.
  • Reminder Emails: Daily reminders for schedule follow-up and study completion.

Daily Database Updates

  • Study Completion: Automatic marking of appointments as Completed based on specific criteria.
  • Family Release: Release families from a lab when associated study appointment is marked as Completed.

New User Interface

A new user interface for scheduling studies. Explore through the DRDB scheduling demo.

Performance Improvements and Bug Fixes

Various system enhancements and bug fixes to improve performance and functionality.
